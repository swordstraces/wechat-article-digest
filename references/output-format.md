# 微信文章摘要输出格式规范

## 设计理念

**漏斗模型** — 从 10 秒 TLDR 到 3 分钟深度价值，逐层展开。

```
TLDR (10秒) → 方法论 (30秒) → 结论 (1分钟) → 观点 (2分钟) → 对我的价值 (3分钟)
```

## 各部分格式要求

### TLDR
- 1-3 句话，总字数 ≤ 80 字
- 必须包含：核心结论 + 关键前提
- 禁止废话前缀（「文章认为」「作者指出」）
- 引用块格式 `>`

### 核心方法论
- 无序列表，4 个维度（方法论/假设/框架/局限）
- 用加粗标记维度名
- 局限性是加分项，不强制

### 核心结论
- 有序列表，3-5 条
- 格式：`**[标题]**：[内容]`
- 必须附带数据支撑表格
- 必须可执行或有明确指向

### 核心观点
- 有序列表，3-5 条
- 每条 ≤ 50 字
- 与结论的区别：观点偏过程性，结论偏结果性

### 关键洞察
- 无序列表，1-3 个
- 如无真正独特的洞察，可省略
- 不要凑数

### 对我的价值
- 三个层次：可直接应用 / 值得深入研究 / 需要验证
- Checklist 格式 `- [ ]`
- 每个价值点必须**具体可执行**

### 标签
- 3-5 个，`#` 开头，空格分隔
- 要具体可搜索

### 元数据
- 表格格式
- 重要程度用 ⭐ 表示（1-5 星）
- 文章类型：宏观报告/行业研究/策略框架/技术分析/观点评论
- 可执行性：高/中/低

## 完整示例

以刚才的广发宏观文章为例：

```markdown
# 📄 文章摘要

**标题**：静待贝叶斯收敛：大类资产配置月度展望
**来源**：郭磊宏观茶座
**作者**：陈礼清
**字数**：49,264
**阅读时间**：123 分钟
**链接**：https://mp.weixin.qq.com/s/D0LubnIv6nXxH63MXZRljw
**保存时间**：2026-04-07

---

## ⚡ TLDR

> 3 月全球大类资产进入「贝叶斯定价」状态（碎片化信息反复博弈，无单边趋势），中国资产彰显韧性。Q1 实际 GDP 5.07%，估值宏观偏离度回落至安全区间，M1-BCI-PPI-CSAD 信号转扩张，倾向于超配权益、4-5 月逆向布局。

---

## 🔬 核心方法论 / 核心假设

- **方法论**：历史复盘 5 次战争/能源危机 +「软数据 vs 硬数据」双轨对比 + 多因子择时体系（M1-BCI-PPI-CSAD）
- **核心假设**：地缘冲击不会长期化；中国资产韧性逻辑持续；流动性冲击已基本释放
- **分析框架**：贝叶斯定价（碎片化信息驱动，缺乏单边趋势）+ 估值-宏观偏离度定位市场
- **局限性**：历史复盘样本仅 5 次、GDP 估算基于调整值而非最终数据、择时模型存在过拟合风险

---

## 📌 核心结论

1. **估值安全边际已明显增加**：全 A 估值宏观偏离度从 +2.01σ 极致位回落至 +1.34σ，后续估值扩张天花板重新打开
2. **4-5 月存在超跌反弹机会**：全球「滞」逻辑不会太快形成，权益资产有逆向布局窗口
3. **CSAD 信号指向风险出清**：羊群因子从 12.4% 分位反弹至 40% 分位，酝酿下一期积极信号
4. **国内资产韧性逻辑确认**：Q1 实际 GDP 5.07%、名义 GDP 5.47%，硬数据积极向上修复
5. **均衡配置优于押注单一场景**：各类资产赔率胜率优势重新获得定价权

### 📊 关键数据支撑

| 维度 | 数据 |
|------|------|
| Q1 实际 GDP | 5.07%（估算） |
| Q1 名义 GDP | 5.47%（估算） |
| 全 A 估值偏离度 | +1.34σ（上期 +2.01σ） |
| 10Y 国债利率 | 处于 56.2% 分位 |
| M1-BCI-PPI-CSAD | 信号转扩张 |
| CSAD 分位 | 12.4% → 40%（风险出清） |

---

## 🎯 核心观点

1. **中东战局超预期**：油价触阈值引发非线性反应，滞胀交易成为结构性主线
2. **贝叶斯定价成主导**：碎片化信息反复博弈，单边趋势显著弱化
3. **流动性冲击广谱传递**：宏观折现率变化传递至股债金汇，跨资产相关性提升
4. **地缘定价进入 2.0**：基本面与产业逻辑定价权上升，低相关方向受益

---

## 💡 关键洞察

- CSAD 衡量的不是「流动性驱动的同涨同跌」，而是市场在抢跑「资产属性漂移」— 后者频率低但具有趋势性
- 全球全天候策略（ALLW ETF）在跨资产对冲性下降后，单日回撤接近 2025 年 4 月初，传统分散化逻辑正被系统性破坏
```
